Verified Bike
================

A simple web application (http://verified.bike) designed to encourage people to not buy stolen bikes online. Bike owners can verify a bike which gives them a simple URL and watermarked images to share in their online marketplace. 

Ruby on Rails
-------------

This application requires:

- Ruby 2.2.3
- Rails 5.0.0.1

Learn more about [Installing Rails](http://railsapps.github.io/installing-rails.html).
